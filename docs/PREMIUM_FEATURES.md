# Catálogo Premium

El catálogo es cerrado y usa las mismas claves en Apps Script, web y Flutter.

| Clave | Función real | Punto de aplicación |
| --- | --- | --- |
| `advancedReminders` | Recordatorios avanzados | Anticipación personalizada en el formulario de compromiso |
| `advancedReports` | Reportes avanzados | Gráfico de distribución por categoría del panel |
| `payoffPlanner` | Planificador de pagos | Sección Plan y generación de estrategia |
| `customPaymentSchedules` | Frecuencias personalizadas | Intervalos cada N días o personalizados |

Premium vigente incluye las cuatro. El administrador puede conceder o retirar cada función manualmente. No se aceptan claves inventadas desde el navegador. Exportación, respaldo, datos básicos, pagos, calendario y recordatorios estándar permanecen disponibles según el alcance Free documentado.
